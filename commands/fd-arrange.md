# /fd-arrange

Trigger the **frontend-design** skill in **arrange mode**.

Use this command when you want to:
- Fix spatial rhythm when elements feel cramped or scattered
- Apply a consistent 4pt grid system across the layout
- Adjust whitespace and breathing room between sections
- Refine visual hierarchy through spatial relationships

## How to Use

```
/fd-arrange
```

Examples:
- `/fd-arrange` on a form where inputs are unevenly spaced
- `/fd-arrange` on a page where sections run into each other without clear separation

## What Happens

1. The frontend-design skill activates in arrange mode
2. The spatial-design reference is loaded for layout best practices
3. A 4pt grid system is applied to all spacing values
4. Whitespace rhythm is fixed between sections and components
5. Visual hierarchy is reinforced through spatial relationships
6. Changes are applied and verified

## Output

A spatially refined layout with consistent 4pt grid spacing, proper whitespace rhythm, and clear visual hierarchy.
