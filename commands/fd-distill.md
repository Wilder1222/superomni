# /fd-distill

Trigger the **frontend-design** skill in **distill mode**.

Use this command when you want to:
- Reduce visual complexity in an overloaded interface
- Remove elements competing for attention
- Apply the "less but better" design principle
- Simplify without losing essential functionality

## How to Use

```
/fd-distill
```

Examples:
- `/fd-distill` on a cluttered settings page with too many controls
- `/fd-distill` on a dashboard where nothing stands out because everything is emphasized

## What Happens

1. The frontend-design skill activates in distill mode
2. Visual noise sources are identified — redundant borders, competing colors, excessive elements
3. Each element is evaluated for necessity and contribution to user goals
4. Removals and simplifications are suggested following "less but better" principle
5. Hierarchy is restored by reducing competing visual signals
6. Changes are applied and verified

## Output

A simplified interface with reduced visual noise and clearer focus on essential content.
