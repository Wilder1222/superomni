# /fd-polish

Trigger the **frontend-design** skill in **polish mode**.

Use this command when you want to:
- Perform a final refinement pass before shipping UI
- Fix micro-details like spacing, alignment, and transitions
- Ensure production-grade quality across all design dimensions
- Catch visual inconsistencies missed during development

## How to Use

```
/fd-polish
```

Examples:
- `/fd-polish` before deploying a new feature page
- `/fd-polish` after a design review to address final nits

## What Happens

1. The frontend-design skill activates in polish mode
2. All 7 design dimensions are checked systematically
3. Micro-details are fixed — spacing, alignment, transitions, border radius, shadows
4. Visual consistency is verified across components
5. Production quality is ensured with pixel-level attention
6. Changes are applied and verified

## Output

A production-ready UI with all micro-details refined and visual inconsistencies resolved.
