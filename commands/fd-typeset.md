# /fd-typeset

Trigger the **frontend-design** skill in **typeset mode**.

Use this command when you want to:
- Fix unclear text hierarchy between headings, body, and labels
- Replace generic system fonts with distinctive typeface choices
- Apply a modular type scale for consistent sizing
- Correct line-height, letter-spacing, and measure (line length)

## How to Use

```
/fd-typeset
```

Examples:
- `/fd-typeset` on a content page where headings and body text look the same weight
- `/fd-typeset` on a UI with default sans-serif that lacks personality

## What Happens

1. The frontend-design skill activates in typeset mode
2. The typography reference is loaded for type hierarchy best practices
3. A modular type scale is applied for consistent heading and body sizing
4. Distinctive, well-paired fonts are selected for the project
5. Line-height, letter-spacing, and measure are corrected for readability
6. Changes are applied and verified

## Output

A refined type system with modular scale, distinctive fonts, correct line-height, and clear heading-to-body hierarchy.
